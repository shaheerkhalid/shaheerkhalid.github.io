UAE Classifieds Platform — System Architecture

Production-grade multi-source classifieds aggregator · Secure-by-design · UAE-hosted

Security Architecture

  • • Argon2id password hashing (OWASP-recommended)
  • • JWT with refresh token rotation (15min access, 7d refresh)
  • • RBAC: Guest → User → Moderator → Admin → Super Admin
  • • CSP, HSTS, X-Frame-Options, CSRF tokens on all routes
  • • Rate limiting: 100 req/min per IP, 1000 req/min per user
  • • Input validation via Zod schemas — all entry points
  • • Audit logging (no PII in logs — passwords/tokens never logged)

Data Flow & Compliance

  • • TLS 1.3 everywhere — no plaintext comms
  • • S3 SSE-KMS for images at rest
  • • PostgreSQL TDE + Row-Level Security (RLS)
  • • Elasticsearch with Arabic text analyzer (UAE bilingual)
  • • robots.txt respected · rate-limited external access
  • • UAE Data Protection Law compliant · Privacy by Design
  • • Data minimization — only collect what's needed

Deployment & DevOps

  • • Docker Compose for local dev (all services)
  • • Kubernetes manifests ready for production (EKS/GKE)
  • • GitHub Actions CI/CD: test → build → scan → deploy
  • • SAST (SonarQube) + DAST (OWASP ZAP) in pipeline
  • • Dependency scanning (Snyk / Dependabot)
  • • Blue-green deployments · automated rollbacks
  • • Prometheus + Grafana dashboards · PagerDuty alerts
